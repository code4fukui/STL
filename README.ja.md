# STLユーティリティ
STL形式のファイルを操作するためのツールキットです。

## デモ
[STL Viewer](https://code4fukui.github.io/stl-util/)で3Dモデルを表示できます。

## 機能
- STLファイルの読み込み、書き出し
- 立方体の自動生成
- Mesh to STLの変換

## 必要環境
Deno 1.34以上

## 使い方
`deno run https://raw.githubusercontent.com/code4fukui/stl-util/main/makeSTLBox.example.js`で立方体を生成できます。
その他の例は`deno run https://raw.githubusercontent.com/code4fukui/stl-util/main/mesh2stl.example.js`などを参照してください。
